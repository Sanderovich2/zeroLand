## О боте

**ZeroLand** - это Telegram-бот, который превращает обычный русский текст в **красивый Unicode-алфавит** (ZeroLand).

Каждая буква заменяется на соответствующий красивый символ:
- **а** → **Δ**  
- **б** → **6**  
- **в** → **β**  
- **г** → **г**  
- **д** → **д**  
- **е** → **ε**  
- **ё** → **ё**  
- **ж** → **ж**  
- **з** → **3**  
- **и** → **u**  
- **й** → **й**  
- **к** → **κ**  
- **л** → **ʌ**  
- **м** → **м**  
- **н** → **н**  
- **о** → **ο**  
- **п** → **π**  
- **р** → **ρ**  
- **с** → **c**  
- **т** → **τ**  
- **у** → **y**  
- **ф** → **φ**  
- **х** → **x**  
- **ц** → **ц**  
- **ч** → **ч**  
- **ш** → **ш**  
- **щ** → **щ**  
- **э** → **э**  
- **ю** → **ю**  
- **я** → **λ**

И так же для заглавных букв (А → Δ, Б → 6, В → Β и т.д.).

Бот **не трогает** `@username` и другие специальные символы.

---

## ✨ Основные функции

1. **Обычный режим** - просто переводит текст
2. **Декоративный режим** - добавляет рамки `⟦ пример текста ⟧`
3. **Inline-режим** - ищи бота в поиске Telegram и пиши `@zeroland_translatebot твой текст`
4. **Команды** `/normal`, `/decorative`, `/alphabet`
5. **Админ-панель** (только для ADMIN_ID) - перерыв, отправка всем, обновление кода

### 1. Требования
- Python 3.8+
- `pip install pyTelegramBotAPI python-dotenv`

### 2. Создай папку и файлы

```bash
mkdir zero-land-bot
cd zero-land-bot
```
